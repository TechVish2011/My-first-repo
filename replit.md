# Number Analyzer

## Overview
A simple and fun Python CLI tool that analyzes numbers and provides mathematical information along with motivational messages.

**Author:** Vishnu Pratap Singh  
**Level:** Beginner Project  
**Language:** Python 3.12

## Project Description
This is a beginner-friendly Python application that:
- Analyzes numbers (even/odd, positive/negative)
- Calculates square and cube values
- Lists factors for integers
- Provides random motivational messages

## Project Structure

## How to Run
The application runs as a CLI (Command Line Interface) tool. Simply run the workflow and enter a number when prompted.
